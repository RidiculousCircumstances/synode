# Synode Fabricator

Fabricator is a local developer workflow for structured multi-expert review before implementation or review handoff. It is not an app-level runtime tool and it does not commit, push, open pull requests, or bypass Synode approvals.

Use `uv run synode fabricator validate` after changing this directory. Use `uv run synode fabricator smoke` for an end-to-end local workflow under `/tmp`.
