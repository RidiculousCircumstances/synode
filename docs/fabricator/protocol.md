# Fabricator Protocol

Fabricator creates a run workspace under `var/fabricator/runs` by default. The Principal Arbiter selects the implementation boundary, receives independent expert responses, synthesizes disagreements, records traceability, and produces a handoff for a human developer.

Fabricator is advisory. It may prepare plans, review reports, and handoff artifacts, but the human operator remains responsible for code changes, verification, commits, and push decisions.
